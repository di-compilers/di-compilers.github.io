Compiler Tools
 


Important tools for course and useful in general

  • JFlex tool for generating lexical analyzers for the Java programming language
    • Website http://jflex.de/
    • Download jflex in compressed form
    • Installation instructions
    • JFlex user manual
    • You can also download it as jflex package for Debian or Ubuntu GNU/Linux (aptitude install jflex)
    • Useful example/tutorial of JFlex specification file to get started with
  • Java CUP for generating bottom-up LALR parsers for the Java programming language
    • Website http://www2.cs.tum.edu/projects/cup/
    • Download page and usage instructions
    • Download it in jar package form sablecc32.jar
    • Java CUP user manual
    • A minimal calculator CUP specification file to get started with
  • JavaCC tool for generating top-down LL(k) parsers and lexical analyzers for the Java programming language
    • Website http://java.net/projects/javacc/
    • Download javacc in compressed form
    • Download it in jar package form javacc5.jar
    • You can download it as javacc package for Debian or Ubuntu GNU/Linux (aptitude install javacc)
    • Documentation and grammar examples inside the zip file
  •  JTB (Java Tree Builder) transforms javacc grammars in Java class hierarchies
    • Website http://compilers.cs.ucla.edu/jtb/
    • Download jtb132 in jar package form
    • You can get our version of the file from jtb132di.jar
    • The above version has some changes compared to the official that you may find useful


Other useful tools:

  • Flex for generating lexical analyzers in C/C++
    • Website (for POSIX systems like Linux, BSD, Unix*) http://flex.sourceforge.net/
    • Implementation of GNU (for POSIX systems and Windows)
    • Manual at http://flex.sourceforge.net/manual/
  • Bison for producing bottom-up LALR(1) parsers in C/C++
    • Website http://www.gnu.org/software/bison/
    • Implementation of GNU (for POSIX systems and Windows)
    • Manual at http://www.gnu.org/software/bison/manual/index.html
  • SableCC for generating bottom-up LALR(1) parsers and lexical analyzers for the Java programming language
    • Website http://sablecc.org/
    • Download sablecc in compressed from
    • Download it in jar package form sablecc32.jar
  • SPIM tool for executing MIPS code
    • Website http://pages.cs.wisc.edu/~larus/spim.html
    • Download it from the above link and for platforms debian/ubuntu gnu/linux download the package "spim"

 

  • Εργαλείο Yacc για την παραγωγή συντακτικών αναλυτών (στο οποίο βασίστηκε το εργαλείο bison)
    • Yacc: A Parser Generator. Άρθρο των S. C. Johnson και R. Sethi στον τόμο Unix Vol. II: research system (10η έκδοση).
      • Πολύ ενδιαφέρον άρθρο! Αξίζει να το διαβάσετε! Ίσως το μοναδικό άρθρο που εξηγεί πολύ καλά τη σχέση των προγραμματιστικών τεχνικών του yacc (άρα και του bison) με τη θεωρία που παρουσιάζεται στο Κεφάλαιο 5 του Dragon Book και συζητήθηκε στην ενότητα του μαθήματος με τίτλο "Μετάφραση οδηγούμενη από το συντακτικό".
